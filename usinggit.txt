1) Download git from (git-scm.com)
2) Create github account from (github.com)
3) Create new github repository
4) go to your project folder and open cmd/gitbash
5) git init
6) git config --global user.name "<your github username>"
7) git config --global user.email "<your github email>"
8) git status
9) git add . or git add -A
10)git remote add origin <your github repo url>
11)git commit-m "<your message>"
12)git push -u origin master
